GUI Skinning Info:
Display:
	DisplayWidth = width of window
	DisplayHeight = height of window

Colors: r,g,b,a (integers between 0 and 255)
	Color1 = Primary color (color of text on-screen typically)
	Color2 = Secondary color (color of window/container borders)
	Color3 = Tertiary color (color of window backgrounds)

Fonts:
	UIFontPath = path to .ttf file for the font used by the GUI
	SmallFontSize = size of 'small' font in GUI (e.g. WindowContainer titlebars)
	MediumFontSize = default size of text labels and button text
	LargeFontSize [unused mostly]
